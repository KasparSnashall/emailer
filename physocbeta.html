How am I doing this?

This sytem runs on open source software in the python langauge its very easy to use and will allow valuable experience in python which is fast becoming a popular language. It uses several python modules such as beautiful soup 4 all installed on a virtual environment using pip. If you have a linux or mac this sytem is simple to set up. It isnt so nice with windows so I would recommend installing linux Ubuntu along side it. I will be more then willing to sit down with thee secretery to show them how to use it and set it up.

Is it easy?

If you know a bit of python then its simple a background in html is also useful but if not it takes about 1 to 2 hours to become an expert in it. Hence you can create lovely emails which will increase the popularity of our society no end. Not to mention you can put up your adverts for talks here. It will also allow you to put up links to scientific articles and other media very easily which I would love to see more of in the society.
